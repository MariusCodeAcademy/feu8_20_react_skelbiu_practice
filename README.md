# Praktika

## SingleAddPage

1. parsiusti ir atvaizduoti konkretau skelbimo informacija
2. prideti add to cart mygtuka

## CreateAdd puslapyje

1. validuoti visus ivesties laukus su firestore rules kad validacija butu tokia pat front end ir firebase backe.
2. isitikinti kad veikia

## Susikurti nauja projekta MyBlog

1. pradeti nauja projekta.
2. prisideti firebase
3. Sukurti BlogsPage, AddBlog, EditBlog puslapius
4. Prisideti routeri
5. padaryti kad galetum sukurti, editinti, istrinti postus
6. posto laukai: title, autorius, imgUrl, body, category, date ir kt
7. BlogsPage atvaizduoti visus postus.
8. BlogsPage isrikiuoti postus mygtuku paspaudimu is front end arba back end pagal date arba title
9. dizaina pasiskolinti is siuolaikiniu pvz
